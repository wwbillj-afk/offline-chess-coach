# Offline Chess Coach

A 100% offline chess learning app inspired by Dr. Wolf.

## Features (Planned)
- Offline chess engine with legal move validation
- Voice-controlled moves (e.g. "Knight to f3")
- Interactive AI-style coach with move-by-move explanations
- Visual feedback for book moves and mistakes
- Local SQLite knowledge base from uploaded chess books
- Hybrid analysis using Stockfish (WebAssembly) and opening books
- Clean, dark, one-handed mobile UI

## Core Principle
No internet required. All logic, assets, and learning data are bundled locally.
